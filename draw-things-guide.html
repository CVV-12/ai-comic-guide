Draw Things 完全使用指南

在 Mac 上本地运行 Stable Diffusion，免费、离线、无限出图

1 下载模型

Draw Things 首次使用需要下载一个生图模型。

推荐模型

模型名风格适合大小
GuoFeng3 推荐 中国古风 / 仙侠 洪荒题材完美适配 ~7GB
DreamShaper 写实 + 幻想 通用场景、人物 ~7GB
Realistic Vision 照片级写实 现代场景、真人质感 ~7GB
1

打开模型下载界面

Draw Things → 顶部中间模型名称 → 点 "Download Models"

2

搜索并下载

搜索框输入 GuoFeng → 找到 GuoFeng3 → 点右侧下载图标。等待下载完成（约 5-10 分钟，取决于网速）

3

切换到已下载的模型

下载完成后点模型名称 → 在列表里选 GuoFeng3 → 模型加载完成，可以开始了

2 生成第一张图

1

设置画幅比例

点底部画幅图标 → 选 16:9（横屏漫剧）或 9:16（抖音竖屏）

2

粘贴 Prompt

把分镜脚本里的英文 Prompt 粘贴到 Positive Prompt 输入框

masterpiece, best quality, endless primordial chaos void,
grey-purple mist surging, no sky no earth, no light,
cosmic scale, distant overhead shot, epic, dark atmosphere
3

粘贴负面 Prompt

把以下通用负面词粘贴到 Negative Prompt 框：

lowres, bad anatomy, bad hands, extra fingers, blurry,
deformed, ugly, watermark, text, jpeg artifacts,
worst quality, low quality, normal quality
4

点击 Generate

底部绿色大按钮。等待 30 秒 ~ 2 分钟（Apple Silicon 芯片越快生成越快）

参数说明（新手保持默认即可）

参数默认值作用
Steps25迭代步数，越高越精细但也越慢。20-30 够用
CFG Scale7Prompt 遵循度，越高越听 Prompt 的话。5-10 之间
Seed-1 (随机)随机种子，点骰子换随机值，同一 Prompt 出不同图
Image Size512×512改成 1024×576（16:9）或 576×1024（9:16）

3 训练白尘 LoRA（锁脸）

漫剧最核心的一步——让白尘在 13 个镜头里长得一样。

生成定妆照用镜头11 Prompt
→
生成多角度图改Seed，跑10-15张
→
全部拖入LoRA训练Trigger: baichen
→
训练完成约5-15分钟
→
生图加 <lora:baichen:0.8>

详细步骤

1

生成白尘定妆照

用镜头 11 的 Prompt，生成一张满意的白尘正面半身像。不满意就多点几次骰子换 Seed 重跑。

masterpiece, best quality, 1boy standing in void,
long black hair to waist, handsome refined face,
faint pearl mark on forehead, bare chested showing lean muscles,
ancient Chinese immortal aesthetic, ethereal lighting,
front view, looking at camera

生成后，长按图片 → "Save to Album"（存到 Draw Things 内置图库）

2

生成多角度参考图

在上面 prompt 基础上，修改视角词，每种视角跑 2-3 张，挑好的存到 Album：

添加的视角词效果
side view, profile侧面
3/4 view, looking left四分之三侧面
looking up, low angle仰视
half body shot半身像
close-up face, front view脸部特写

总共凑 10-15 张 同角色不同角度的高清图

3

开始 LoRA 训练

Draw Things 顶部 → "LoRA" 标签 → 点 "Create LoRA"
把所有白尘图从 Album 拖进训练界面
Trigger word 填：baichen
其他参数保持默认 → 点底部 "Train"
训练时长：M1 ≈ 15分钟，M2/M3 ≈ 8分钟，M4 ≈ 5分钟

4 使用 LoRA 生成角色图

1

开启 LoRA

在画图界面点 "LoRA" 开关 → 找到 baichen → 拖动滑块到 0.7 - 0.8

2

Prompt 加 Trigger Word

在所有需要白尘出场的 Prompt 里，最前面加上 baichen：

baichen, masterpiece, best quality, 1boy,
long black hair, young Taoist disciple,
green robe, kneeling in celestial palace...

LoRA 强度参考

强度效果适用场景
0.5 - 0.6轻参考，有一定自由度远景、背影、小比例人物
0.7 - 0.8标准参考，脸型固定中景、半身、大部分镜头
0.9 - 1.0强锁定，几乎复制训练图脸部特写镜头

5 逐镜头生成提示

13 个镜头的生图策略，哪些开 LoRA，哪些直接生成。

镜头内容LoRA强度备注
01混沌虚空 不需要-纯场景，直接生成
02沉睡盘古 不需要-场景+盘古，无白尘
03白尘意识体 开启0.7第一张白尘出镜，是关键图
04系统面板+白尘 开启0.7过肩镜头，LoRA保证侧脸一致
05混沌珠特写 不需要-物品特写
06白尘伸手夺珠 开启0.7侧面+手部动作
07金光护体 开启0.6金光可能盖住脸，强度略降
08握住混沌珠 开启0.8需要脸部+手部都清晰
09盘古开天 不需要-大场景，白尘只是小光点
10白尘变形融合 关闭-变形中的脸不适用LoRA
11白尘新身体 开启0.8最重要的定妆镜头！
12拜师通天 开启0.7白尘跪拜，只对白尘加LoRA
13白尘脸部特写 开启0.9收尾大头特写，高LoRA强度

6 常见问题 & 技巧

图不满意怎么办

  • 换 Seed：点 Generate 按钮旁边的骰子图标，同样的 Prompt 能出完全不同但都好看的图
  • 增加 Steps：从 25 改到 35-40，画面更精细（但也更慢）
  • 调整 CFG：如果画面"不听话"，提到 8-9；如果画面"太死板"，降到 5-6
  • Img2Img 修补：找到一张"还行但不够好"的图，拖进画布，切换到 Img2Img 模式，Strength 设 0.3-0.5，重新生成

手画崩了 / 手指多了

这是 SD 的通病。三个解决方案：

  • 负面 prompt 里确保有 bad hands, extra fingers, mutated hands
  • 多跑几张，选手部正常的（最省事）
  • 用 Inpaint 功能：画笔涂掉问题区域，单独重绘那一小块

角色老是像另一个人

  • 确认 LoRA 开关已打开 + 强度 ≥ 0.7
  • 确认 Prompt 最前面有 trigger word baichen
  • 如果还是不像，把 LoRA 强度提到 0.9
  • 如果换个角度就不像，说明训练素材角度不够多，回到步骤 3.2 补充更多角度图

生成太慢

  • 降低分辨率：不要一上来就 1920×1080，先用 768×432 出小图，满意后再用 Upscale 放大
  • 降低 Steps：20 步其实已经够用于筛选，最终图再用 30 步
  • M1/M2 Mac：耐心等待，一张 1024×576 大约 1-3 分钟
先在线试，再本地产：建议先用 liblib.ai 在线生成几张开开眼，确认 Prompt 效果好，再用 Draw Things 本地批量生成 + LoRA 锁脸。省时又保险。
